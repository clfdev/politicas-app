Política de Privacidade do app Astronomia: Terra e Lua

Data: 10/05/2025

Esta é a política de privacidade do aplicativo ASTRONOMIA: TERRA E LUA. Explicamos aqui como seus dados são tratados.

1. Coleta de Dados

Esse aplicativo não coleta nenhum tipo de dado. Ele não possui backend conectado a banco de dados para armazenar dados. O aplicativo roda apenas no frontend.

2. Uso de Dados

Como nenhum dado é coletado, também não há uso ou tratamento de informações pessoais.

3. Compartilhamento de Dados

O aplicativo não compartilha nenhum dado com terceiros, pois nenhuma informação é coletada.

4. Permissões do Aplicativo

Este aplicativo não solicita permissões de acesso a recursos do dispositivo, como câmera, localização ou arquivos.

5. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta página com frequência para estar ciente de qualquer alteração.
